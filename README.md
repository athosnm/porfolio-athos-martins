## Português

Portfolio pessoal desenvolvido com HTML, CSS e JavaScript para apresentar minha trajetória, tecnologias, projetos e formas de contato.

### Sobre o projeto

O site foi criado como uma página responsiva e bilíngue, com foco em uma apresentação objetiva para oportunidades, networking e evolução profissional em desenvolvimento backend.

### Funcionalidades

- Layout responsivo para desktop e mobile
- Alternância entre português e inglês
- Menu mobile acessível
- Tela de carregamento com progresso
- Animações suaves ao rolar a página
- Cursor personalizado em dispositivos com mouse
- Botão para copiar email
- Links para GitHub, LinkedIn e contato por email

### Tecnologias

- HTML5
- CSS3
- JavaScript
- Git
- GitHub
